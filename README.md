## Instructions to Run game:

> python3 game.py

This will open game with waiting pad.

### Keys involved

    'a' : move paddle to left
    'd' : move paddle to right
    'w' : to activte ball
    'q' : to quit anytime

#### Note: if game is not loading properly please clear "__pycache__" and then start.
